## 如何使用

1. 打开 `index.html` 文件启动主控制器
2. 使用控制按钮或键盘快捷键控制场景播放：
   - 空格键：播放/暂停
   - 左箭头：上一个场景
   - 右箭头：下一个场景
   - F键：全屏切换
3. 点击右侧场景列表可直接跳转到指定场景
4. 使用底部进度条查看当前播放进度
5. 可调节音量控制（如场景包含音频元素）

## 技术实现

- 前端技术：HTML5, CSS3, JavaScript
- 动画效果：Canvas API, CSS动画
- 响应式设计：适配不同屏幕尺寸
- 场景管理：iframe嵌入，独立场景文件

## 设计理念

本项目通过37个精心设计的场景，将Mili的《World.execute(me)》歌曲中的情感和叙事转化为视觉体验。每个场景都对应歌曲中的特定部分，通过视觉元素、动画和交互来传达歌曲的核心主题：存在、意识、自由与囚禁、爱与执行。

## 场景主题

1-14场景：执行与重复的循环
15-17场景：被困在模拟中的认知
18-23场景：身份与存在的探索
24-30场景：痛苦与绝望的体验
31-37场景：爱与救赎的追寻
